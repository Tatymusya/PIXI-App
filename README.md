# PIXI-App

Beautiful game app on developing with Vue 3 (TypeScript), Pixi.js in Vite.

## Project Setup

### Clone project

```sh
git clone (@url)
```

### Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

## Project Setup (Docker)

### Build and launch

```sh
docker build -t pixi-app .
docker run -d -p 8080:80 --name my-pixi-app pixi-app
```
Open: http://localhost:8080
